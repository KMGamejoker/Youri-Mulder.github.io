Over Mij

Ik ben Youri Mulder en ik vind het geweldig om games te maken. Het maakt mij over het algemeen niet heel veel uit aan wat voor soort game ik werk, waar ik de plezier uithaal is een systeem bedenken om een feature te maken en dat vervolgens ook werkent te krijgen. Ik volg nu de opleiding game development op het GLU (Grafish lyseum Utrecht) en zit in jaar 2. naast het ontwikkelen van games vind ik het uiteraard ook heel leuk om zelf te gamen. zelf speel ik voornamrlijk veel rythem games maar shooters speel ik ook best veel. naast het binnen zitten en bezig zijn met games ben ik ook regelmatig buiten met vrienden. Wat mij de inspiratie heeft gegeven om game developer te worden was de end cutscene van ark extintion, het leek me zo vet om een game te maken met een heel verhaal te maken of onderdeel te zijn van de productie van een grote game. ook de game developer youtuber dani heeft mij inspiratie gegeven nadat ik sowiso al wist dst ik game developer wilde worden. Mijn eerste ervaren met game development waren met Gamemaker Studio en Roblox Studio. In Gamemaker Studio hsd ik toen een player werkent gekregen met drag boxes en in Roblox gebruike ik voor het eerst code met hulp van een tutorial video. Later ben ik gaan rondkijken in Unity en een jaar later gestart op het GLU
